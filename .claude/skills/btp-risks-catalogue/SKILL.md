# Catalogue Risques BTP — Genève

## Risques par probabilité (données terrain GE)

### 🔴 Très fréquents (> 30% des chantiers)
| Risque | Fréquence | Surcoût moyen | Délai additionnel |
|--------|-----------|--------------|-------------------|
| Client modifie les plans en cours | 40% | +20% HT | +1–3 sem |
| Humidité / infiltration cachée | 30% | +CHF 5–15k | +2 sem |
| Accès chantier difficile / non prévu | 35% | +CHF 2–5k | +3j–1 sem |
| Plages horaires restreintes (copropriété) | 30% | +15% MO | +1–2 sem |

| Matériau en rupture de stock | 20% | +CHF 0–3k | +1–4 sem |

| Plomb dans peintures (bâti avant 1970) | 20% | +CHF 2–8k | +1 sem |

| Risque | Fréquence | Surcoût moyen |
|--------|-----------|--------------|

## Risques légaux et administratifs GE

### Permis et autorisations
```
Sans AC/PT → amende + démolition → toujours vérifier avant démarrage
Travaux amiante sans ORCAB → amende pénale + responsabilité civile
Travaux de nuit sans autorisation → CHF 500–5000/nuit
Non-conformité aux normes NIN (électricité) → mise hors service
```

### Responsabilités
```
RC professionnelle : obligatoire pour tout entrepreneur GE
Garantie décennale : 10 ans sur les vices cachés structurels (art. 371 CO)
Délai de réclamation apparente : 2 ans (SIA 118 §172)
Sous-traitance > 20% : déclarer au maître d'ouvrage (CCT)
```

## Risques saisonniers Genève

```
Janvier–Février  : gel → arrêt travaux extérieurs, mortiers non posables
Août             : 3 semaines de vacances secteur BTP → planifier en amont
Décembre         : fériés nombreux, réductions de journées
Canicule (> 35°) : IDEX > 40 → arrêt obligatoire si recommandation OCIRT
```

## Provisions recommandées par type de chantier

| Type de chantier | Provision imprévus |
|-----------------|-------------------|
| Rénovation légère (peinture, revêtements) | 5% HT |
| Second œuvre standard | 10% HT |
| Rénovation bâtiment 1970–1990 | 15% HT |
| Rénovation bâtiment avant 1970 | 20–25% HT |
| Bâtiment classé / protégé MH | 25–30% HT |
| Chantier en site occupé | +5% supplémentaire |
| Data center / salle informatique | +8% (contraintes techniques) |

## Checklist pré-chantier CYNA
- [ ] AC ou PT obtenu (pas juste déposé)
- [ ] Diagnostic amiante si bâtiment > 1990
- [ ] Diagnostic plomb si bâtiment > 1970
- [ ] Accès chantier confirmé (horaires, ascenseur, parking)
- [ ] Assurance RC professionnelle à jour
- [ ] Sous-traitants assurés vérifiés
- [ ] Provision imprévus incluse dans le devis
- [ ] Client solvable vérifié (si marché > CHF 50k)
