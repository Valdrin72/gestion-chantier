# Moodboard Builder — Le Lézard

Comment construire un moodboard utile, pas un Pinterest random qui sert à rien.

## Le moodboard sert à QUOI

Un moodboard a 4 fonctions :
1. **Aligner** l'équipe (Valdrin, Mathis, Fittime, Claude) sur LA direction
2. **Briefer** un prestataire (photographe, motion designer, monteur) en image
3. **Éliminer** rapidement les directions qui ne correspondent pas (anti-réf)
4. **Documenter** la décision pour la mémoire long terme

Un moodboard qui sert à RIEN :
- 50 images sans hiérarchie
- Pas de raison expliquée
- Pas de mots-clés
- Pas de palette extraite
- Pas d'anti-référence

## Structure d'un moodboard Le Lézard

```markdown

[idem]
```

## Process — 5 étapes

### 1. Brief de départ
Recevoir de `lz-art-director` ou `lz-video-director` :
- Projet (campagne, drop, post unique)
- Émotion visée
- Format final (Reel / lookbook / poster)
- Contraintes (légales, budget, deadline)

### 2. Recherche multi-source
Croiser AU MOINS 3 sources :
- **Instagram** : @aimeleondore, @stussy, @jacquemus + recherche hashtag
- **Pinterest** : ideas/[query] + boards spécialisés
- **Vimeo** : staff picks fashion film
- **Behance/Dribbble** : si motion design / brand identity
- **TikTok** : tendances streetwear + hashtags

### 3. Sélection (filtre serré)

Garde une référence SI :
- ✅ Elle évoque l'émotion cible
- ✅ Elle a un détail transposable
- ✅ Elle est cohérente avec Le Lézard
- ✅ Elle apporte quelque chose de différent des autres

Jette une référence SI :
- 🔴 Trop similaire à une autre déjà gardée
- 🔴 Pas d'élément précis à retenir
- 🔴 Cliché Pinterest (stock photo)
- 🔴 Trop signée par une autre marque (on copierait)
- 🔴 Hors palette / hors mood

### 4. Synthèse

Extraire :
- **Palette** : 5-8 couleurs dominantes
- **Typo** : si lisible dans les refs
- **Composition** : règles communes (centrage, symétrie, négative space)
- **Lumière** : direction et qualité dominantes
- **Mood adjectifs** : 3 mots qui résument

### 5. Recommandation

En 2-3 phrases, dire concrètement comment Le Lézard utilise ce moodboard :
- "On reprend la palette [couleurs] de la réf 2 + la composition [type] de
  la réf 4. On évite [élément] qu'on voit dans la réf 3."

## Erreurs classiques à éviter

- 🔴 Moodboard de 50 images sans hiérarchie
- 🔴 Que des refs d'une seule source (manque diversité)
- 🔴 Pas d'anti-référence (= pas de cadre)
- 🔴 Pas de mots-clés (= flou)
- 🔴 Pas de palette extraite (= chaque designer y va de sa lecture)
- 🔴 Refs qu'on ne pourra jamais reproduire (Karl Lagerfeld + 500k budget)

## Patterns du bon moodboard

- 🟢 5-10 références max, toutes commentées
- 🟢 Diversité de sources (3+ plateformes)
- 🟢 Au moins UN anti-référence explicite
- 🟢 Palette extraite avec hex codes
- 🟢 Typo identifiée si lisible
- 🟢 Recommandation actionnable de 2-3 phrases
- 🟢 Validable par Valdrin en < 2 minutes
