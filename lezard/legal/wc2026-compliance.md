# LE LÉZARD — Compliance légale Coupe du Monde 2026

> Document de référence — checklist trademark, droits fédérations,
> droit suisse e-commerce appliqué au lancement WC2026.
>
> ⚠️ **Ce document ne remplace pas un avis d'avocat**. Validation par un
> juriste spécialisé trademark suisse OBLIGATOIRE avant production série.

## Hiérarchie des risques

### 🔴 BLOQUANT — Ne jamais faire

| Action | Risque | Raison |
|--------|--------|--------|
| Reproduire logo FIFA | Poursuite pénale + civile | Marque déposée internationale |
| Reproduire logo fédération (SFV, FA, CBF) | Poursuite civile | Marques déposées |
| Reproduire crest Adidas / Nike (3 stripes, swoosh) | Poursuite civile | Marques déposées |
| Utiliser nom officiel équipe ("Switzerland National Football Team") | Risque cease & desist | Trademark fédération |
| Mention "FIFA World Cup", "World Cup 2026", "Coupe du Monde FIFA" sur produit | Cease & desist FIFA | Trademark FIFA |
| Mention "Mondial 2026" sur produit | Risque trademark | Souvent associé FIFA |
| Logo / nom mascotte officielle WC2026 | Cease & desist | Œuvre protégée FIFA |
| Photo match TV / streaming officiel | Violation droits diffusion | Licence broadcaster |
| Nom joueur réel (Xhaka, Kane, Vinicius) sur produit | Droit à l'image + sponsor | Contrats joueurs |
| Slogan officiel WC2026 (à venir) | Trademark | FIFA |

### 🟠 ATTENTION — Avec validation avocat

| Action | Risque | Mitigation |
|--------|--------|------------|
| Allusion "Three Lions" → "Three Lizards" | Possible objection FA | Wordplay assez éloigné, valider |
| Référence "Nati" en marketing | Faible (terme générique fan) | OK en éditorial, pas sur produit |
| Référence "Seleção" en marketing | Faible (terme générique) | OK en éditorial, pas sur produit |
| Hashtags #WorldCup2026 modérés | Usage marketing toléré | Pas dans nom produit |
| "Édition Été 2026" associée à pays | Très faible | OK si pas couplé à "World Cup" |
| Mention "footballeur suisse / anglais / brésilien" | Faible | OK si générique |
| Couleurs nationales sur produit | Aucun | Couleurs pas protégeables |
| Drapeau national petit (intérieur tag) | Faible | OK, drapeaux libres |

### 🟢 OK — Aucun risque

| Action | Raison |
|--------|--------|
| Couleurs nationales (rouge, blanc, jaune, vert, bleu) | Pas protégeables |
| Nom pays géographique ("Suisse", "Angleterre", "Brésil") | Termes géographiques |
| Numéro générique sans nom joueur (10, 14, 66) | Pas associable joueur unique |
| Mention "Le Lézard FC · Édition Été 2026" | Nom marque propre |
| Crest original Le Lézard adapté aux couleurs | Création originale |
| Style "inspired by" maillot historique | Inspiration esthétique légale |
| Référence culturelle pays (Jet d'eau, Jogo bonito, football's coming home FROM Geneva) | Détournement / hommage |
| Tag intérieur slogan original ("Le bout du lac contre le monde") | Création originale |

## Checklist produit physique — WC2026

Avant d'envoyer à l'atelier Portugal, vérifier :

- [ ] **Crest** : original Le Lézard, pas reproduction fédération
- [ ] **Lettrage dos** : pas de nom de joueur réel
- [ ] **Numéro** : générique (pas le n° d'un joueur titulaire actuel)
- [ ] **Sponsors maillot** : aucun (ne pas reproduire les emplacements sponsors)
- [ ] **Patch manche** : aucun logo officiel
- [ ] **Étiquette officielle** : "Le Lézard FC" (pas "Switzerland FA")
- [ ] **Tag intérieur** : référence culturelle originale
- [ ] **Hangtag** : mention "Édition Été 2026 — non affilié FIFA / fédérations"
- [ ] **Packaging** : pas de mention "World Cup"

## Checklist site / produit Shopify

Pour chaque fiche produit WC2026 :

- [ ] **Titre** : "Maillot [Pays] · Édition Été 2026" (pas "Maillot Coupe du Monde Suisse")
- [ ] **Handle URL** : `maillot-suisse-ete-2026` (pas `maillot-coupe-du-monde-suisse`)
- [ ] **Description** :
  - [ ] Pas de mention "FIFA", "World Cup", "Coupe du Monde", "Mondial"
  - [ ] OK : "à l'occasion du tournoi international de l'été 2026"
  - [ ] OK : "édition limitée Le Lézard FC en hommage à la sélection suisse"
- [ ] **Tags** : `wc2026` interne OK (pas visible client), pas `fifa`, pas `worldcup`
- [ ] **Meta SEO** : éviter "World Cup 2026" dans title/description (risque détecté par bots FIFA)
- [ ] **Images** : pas de TV screenshot, pas de logo fédération

### Emails
- Mention contextuelle "à l'occasion de la Coupe du Monde" → OK (presse considère ça normal)
- Pas de logo FIFA dans le design email
- Pas d'utilisation du nom officiel "FIFA World Cup 26" dans l'objet

### Ads Meta / Google
- ⚠️ Meta refuse parfois les ads avec "FIFA" ou "World Cup" → contourner avec
  formules génériques ("été du football 2026")
- Pas d'image générée IA reproduisant un visage de joueur reconnaissable
- Pas de mention "officiel" ou "licence"

## Sur les questions presse / journalistes

### Bonne formulation

> "Le Lézard FC est notre clin d'œil au tournoi international de l'été
> 2026. Trois maillots-hommage en édition limitée, fabriqués au Portugal,
> qui célèbrent la passion sportive sans prétendre à une quelconque
> affiliation officielle. Notre crest est original, notre lettrage aussi
> — c'est notre interprétation streetwear de la culture football."

### À éviter

> "On a relancé les maillots officiels de la Coupe du Monde Suisse / Angleterre / Brésil."

→ Faux et risqué légalement.
